FoE Datenbank

Diese Datenbank ermöglicht das auswerten verschiedener Informationen aus dem Spiel Forge of Empire.
Daten können als .csv Datei eingepflegt werden. Derzeit bietet die Datenbank unterstützung für folgende Dateien:

Members***.csv
GuildTreasury***.csv
GreatBuildings***.csv

Diese Dateien können aus dem Spiel direkt, oder über den FoE-Helper heruntergeladen werden. Man kann nach einfügen der Daten verschiedene Informationen Auswerten:

Spielerdaten
<img width="1097" height="1014" alt="image" src="https://github.com/user-attachments/assets/f1f6cece-298f-4634-9d65-702626a60164" />
Zeigt neben den regulären informationen auch alle Kassenvorgänge an (Produktionen und Spenden), sowie ein Aktivitätslog der letzten 14 Tage. Man kann sich auch schnell einen Überblick beschaffen, welche Legendären Gebäude der Spieler hat.

Gildenkasse
<img width="1203" height="958" alt="image" src="https://github.com/user-attachments/assets/cc833196-cb13-48a9-b3e5-babf63e659f0" />
Eine Übersicht über alle Spieler und deren Produktion, Spenden sowie verursachte Kosten. Die Berechnung der Kosten ist wie folgt:
Rohstoffzahlung in der Gildenkasse / alle Spieler in dem Zeitalter dem der Rohstoff zugeordnet ist.
Hierbei ist wichtig zu beachten das die Rechnung derzeit nur das aktuelle Zeitalter des Spielers berücksichtigt. Sollte ein Spieler in dem angezeigten Zeitraum das Zeitalter gewechselt haben, oder die Gilde verlassen haben, stimmen die Daten nichtmehr exakt.
Es ist aber meiner Meinung nach, dennoch ein guter Richtwert.

Die Datei foe_gildenkasse_*.accdb ist für user gedacht die Microsoft Access besitzen. Für alle anderen ist die standalone variante. Hierfür wird lediglich eine frei verfügbare Runtime von Microsoft benötigt.
